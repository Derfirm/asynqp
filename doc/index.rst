``asynqp``
==========
An AMQP (aka `RabbitMQ <http://www.rabbitmq.com/>`_) client library for :mod:`asyncio`.

Installation
------------

:mod:`asynqp` has no dependencies outside of the standard library. To install the package:

::

    pip install asynqp
